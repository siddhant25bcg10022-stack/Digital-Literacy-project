# Digital-Literacy-project
This repository contains my Digital Literacy Portfolio Project created as part of the CSE0001 – Digital Literacy course at VIT Bhopal University. The project is designed to develop essential digital skills and promote responsible, safe, and professional online behavior


📌 Project Overview
This project is created as part of the Digital Literacy course. It demonstrates understanding of digital tools, online platforms, professional communication, and cyber safety. The project is divided into five tasks, each focusing on a specific aspect of digital literacy.
